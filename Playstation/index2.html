    • How to

    Plug in your Flash drive, create a new folder on the root of the flash drive called "PS3", inside the folder make another folder called "UPDATE". Now unzip the zip you downloaded into the update file it should end up being called "PS3UPDAT.PUP", then safely eject the USB and plug it into your PS3. Just navigate to Settings tab, click system update, chose update via media storage, it should say found "Version 4.66-JB" if it doesn't say exactly that repeate all the steps! If it says that click Ok and then Accept Conditions and Install Update MAKE SURE IT DOESN'T LOOSE POWER! After a minute you should hear 4 beeps then it turns off, power it on by the console and you're done!
